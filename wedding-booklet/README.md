# 両家顔合わせ しおり テンプレート

結婚の顔合わせ用の、印刷できるしおりテンプレートです。A4縦・全8ページ構成。

## 収録ページ

1. 表紙
2. ご挨拶
3. 当日の流れ（スケジュール）
4. 二人のプロフィール
5. 両家の家族紹介（写真枠付き）
6. 二人の思い出（出会い〜プロポーズ）
7. 今後の予定（入籍・新居・結婚式）
8. 感謝のメッセージ

## 使い方

1. `kaoawase-shiori.html` をブラウザ（Chrome / Edge / Safari）で開く
2. **茶色の文字**と点線の欄を、おふたりの内容に書き換える
   - HTMLを直接編集するか、いったん印刷用に流し込んでから手書きでもOK
3. 右上の「印刷 / PDF保存」ボタン、またはブラウザの印刷から出力
   - 用紙：A4／余白：なし
   - 「背景のグラフィック」を有効にすると、金の枠や色が綺麗に出ます
   - 「PDFに保存」を選べばPDF化できます

## 写真の入れ方

「写真をここに」の枠（`.photo`）を、画像に差し替えられます。

```html
<div class="photo"><img src="family.jpg" alt="" style="width:100%;height:100%;object-fit:cover;"></div>
```

## 冊子にするには

- 中綴じ冊子印刷に対応したプリンタなら、A5冊子として出力できます
- 家庭用プリンタの場合は、A4で印刷して二つ折り・重ねてホチキス留めでも作れます

書体はお使いのPCの明朝体・ゴシック体を使用しています（追加インストール不要）。
