# 両家顔合わせ しおり テンプレート

結婚の顔合わせ用の、印刷できるしおりテンプレートです。A4縦・全8ページ構成。

## 収録ページ

1. 表紙
2. ご挨拶
3. 当日の流れ（スケジュール）
4. 二人のプロフィール
5. 両家の家族紹介（写真枠付き）
6. 二人の思い出（出会い〜プロポーズ）
7. 今後の予定（入籍・新居・結婚式）
8. 感謝のメッセージ

## 使い方（ブラウザで直接入力できます）

1. `kaoawase-shiori.html` をブラウザ（Chrome / Edge / Safari）で開く
2. **茶色の文字**や点線の欄を**クリックして、そのまま入力**する
   - 入力内容はそのブラウザに**自動保存**されます（閉じても消えません）
   - 最初からやり直したいときは「内容をリセット」ボタン
3. 右上の「印刷 / PDF保存」ボタン、またはブラウザの印刷から出力
   - 用紙：A4／余白：なし
   - 「背景のグラフィック」を有効にすると、金の枠や色が綺麗に出ます
   - 出力先を「PDFに保存」にすればPDF化できます

> 補足：自動保存は「開いたブラウザ・端末ごと」です。別のPCに移す場合は、
> 完成後にPDF保存しておくと確実です。じっくり書き換えたい場合は、
> HTMLをテキストエディタで直接編集してもかまいません。

## 写真の入れ方

家族紹介の「写真」枠（`.mphoto`）は、一人ずつ画像に差し替えられます。
`写真` の文字を消して、その中に `<img>` を入れてください。

```html
<div class="mphoto"><img src="father.jpg" alt=""></div>
```

画像は枠に合わせて自動でトリミング表示されます。差し替えが難しい場合は、
印刷後にL判・証明写真などを枠に貼り付けてもOKです。

- 新郎家：父・母・姉・兄
- 新婦家：父・母・弟

## 冊子にするには

- 中綴じ冊子印刷に対応したプリンタなら、A5冊子として出力できます
- 家庭用プリンタの場合は、A4で印刷して二つ折り・重ねてホチキス留めでも作れます

書体はお使いのPCの明朝体・ゴシック体を使用しています（追加インストール不要）。
